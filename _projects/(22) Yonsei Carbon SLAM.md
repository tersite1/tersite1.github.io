## Forest Carbon Estimation via LiDAR-Inertial SLAM

Graduation project. I built an end-to-end LiDAR-inertial SLAM pipeline based on DLIO, calibrated a Livox Mid-70 LiDAR with an IMU, and used the reconstructed point cloud to estimate forest carbon sequestration.
